Other Work

I also have many other projects I've done for school. Unlike the above projects these were done in concert with lectures and project specs. Still, they are a good representation of the fields I studied while getting my degree.

Instagram Clone

A clone of Instagram made for a Web Development class using Flask, React, and a Python backend with SQLite3. Includes likes, comments, follows, making/deleting accounts, profile pages, and a main feed with infinite scrolling.

Inverse Kinematics Simulator

A forward and inverse kinematic calculator written in Javascript for an Autonomous Robotics class. Recursively calculates FK and IK to work with any any robot. Seen here automatically moving to randomly generated setpoints in sequence.

Search Engine

A search engine for a subset of Wikipedia pages built for a Web Development class. Results returned based on TF-IDF and PageRank scores. TF-IDF scores were calculated via MapReduce with an input of raw Wikipedia article text.

Panorama Stitcher

A panorama stitcher built in Python for a Computer Vision class. Identifies image features using ORB to match similar points between images, then utilizes RANSAC to estimate a homography matrix between the extracted features. The homography is then used to apply a perspective transform to merge the two images.

Databases

An ER diagram designed for a Facebook-style database. Entries were moved from the original single-table database to the newly designed database with a series of SQL queries. Other projects included MongoDB and implementing a Grace Hash Join in C++.
